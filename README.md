# Run this on an Android Virtual Device
## 1. Setup virtual device:
- Open AVD Manager => Create Virtual Device
- Select Phone => 5.4" FWVGA => Clone Device...
- Set resolution to 240x240
- Choose a system image
- Finish
## 2. Run
